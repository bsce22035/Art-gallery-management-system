# Art Gallery Application

## Overview
The Art Gallery Application is a full-stack platform designed to bring the world of art closer to audiences. It provides a seamless experience for both artists and customers, allowing artists to showcase their work and customers to explore, participate in virtual exhibitions, and purchase art pieces securely.

## Tech Stack
Frontend: React, CSS, JavaScript, HTML
Backend: Node.js, MySQL
## Key Features
- Artist Management:
Artists can register, manage profiles, and display their artwork.
- Customer Experience:
Customers can browse artwork, join virtual exhibitions, and make secure purchases.
- User-Friendly UI:
Designed for easy navigation and visually engaging art experience.
- Secure Transactions:
Ensures safe and reliable purchase process for art pieces.
